## What is Strimzi?

Strimzi is a CNCF sandbox project that provides a Kubernetes Operator for running Apache Kafka on Kubernetes. It simplifies:

- Kafka cluster deployment and management
- Configuration through Kubernetes Custom Resources
- Automated rolling updates and scaling
- Security with TLS and authentication

## Prerequisites

This tutorial assumes basic knowledge of:
- Kubernetes concepts (pods, deployments, services)
- Apache Kafka fundamentals (topics, producers, consumers)
